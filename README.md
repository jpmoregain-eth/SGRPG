# SGRPG - Singapore RPG

A linear text-based RPG demo set in a supernatural Singapore.

## Play Now

Open `index.html` in any browser. No server needed.

## Story

The Merlion's Tear — a crystal protecting Singapore's prosperity — has been stolen. You must journey from Changi Village to the CBD, battling MRT wraiths, fashion zombies, and ultimately confronting The Collector to save the island.

## Features

- Linear narrative with branching choices
- HP, gold, and inventory system
- Multiple paths and endings
- Singaporean locations and culture
- Typewriter text effect
- Dark terminal aesthetic

## How to Play

1. Open `index.html` in your browser
2. Read the story text
3. Click choices to progress
4. Manage your HP and inventory
5. Reach one of the endings

## Endings

- **Victory (Combat)** — Defeat The Collector in battle
- **Victory (Peaceful)** — Convince The Collector to surrender
- **Victory (Journal)** — Use the Ancient Journal's magic
- **Game Over** — Fall in combat
